CSS

Posted on 2020-04-17

《精通CSS——高级Web标准解决方案》读书笔记

选择符：

  • 匹配某些字符开头的属性值，在等号前面加上插入符（^）

  • 匹配某些字符结尾的属性值，在等号面前加上美元符号（$）

  • 匹配包含某些字符的属性值，在等号前面加上星号（*）

  • 匹配以空格分隔的字符串中的属性值，在等号前面加上波浪号（~）

  • 伪类的先后次序：:link和:visited应该排在前面，然后才是与用户交互相关的。

  • 伪类:target匹配一个有ID属性的元素。

  • nth-child可以交替为表格行应用样式。

  • 有许多针对type值的伪类，这些伪类可以反映表单控件的某种状态。

特殊性：

  • 层叠机制的重要级别从高到低：
    1. 标注为！Import的用户样式
    2. 标注为！Import的作者样式
    3. 作者样式
    4. 用户样式
    5. 浏览器（或用户代理）的默认样式
  • 任何选择符的特殊性都对应如下4个级别，即a,b,c,d：
    1. 行内样式，a为1
    2. b等于ID选择符的数目
    3. c等于类选择符、伪类选择符及属性选择符的数目
    4. d等于类型选择符和伪元素选择符的数目
      选择符的特殊性可能会使我们需要特殊突出的内容被覆盖，因此我们需要尽量提高特殊性，例如给我们需要的组件上加上更厉害的特殊符。或者从一开始简化选择符，降低特殊性。

为文档应用样式：

  • 把样式放在<style>元素中

  • 用link引用外部CSS

  • 还可以在<style>中使用@import指令加载外部CSS文件

  • 使用link或style添加多个样式表或样式块时，声明的次序就是出现的次序。

可见格式化模型：

  • Box-sizing默认值为content-box，若设置为border-box时，此时的width属性是元素所有可见部分的宽度。

  • 如果给类内容里添加内边距，可以设置一个不同的box-sizing值修改盒子宽度的计算方式。

  • 块级盒子会沿垂直方向堆叠，盒子在垂直方向上的间距又它们的上，下外边距决定的。行内盒子是沿文本流水平排列的，也会随文本换行而换行，它们之间的水平间距可以用过水平方向的内边距，外框和外边距来调节。

网页排版：

  • 几乎所有浏览器中font-size的默认大小都是16px
    Font-weigt设置文本粗细，可以使用关键字也可以用数字值，都是一百的整数倍，最大为900。

  • 主体内容的文本行长通常是45~75个字符，平均值为66个字符。
    使用相邻组合符设置text-index属性如：p+p{text-index：}增加观看效果。
    手工断词换行可用&shy软连字符。

  • 自动连字功能

    1. 在html元素中设置语言代码<hrml lang=”en”>
    2. 将相关元素的hyphens属性值设为auto。

  • 跨栏操作：让某些元素流出文本外，如文章标题和最后一段。可用column-span:all以关闭跨栏特性。

  • @font-face规则通过它可以指定浏览器下载Web字体服务器地址，以及如何在样式表中引用该字体。可以接受以下声明：

    1. font-family:必需，字体族的名称
    2. src：必需，URL或URL列表，用于下载字体
    3. font-weigth:可选的字体粗细，默认值为normal
    4. font-style：可选字体样式，默认值为normal

漂亮的盒子：

  • 添加背景颜色再添加图片背景以防图片加载失败。

图片格式：

  1. JPEG：不支持透明设置
  2. PNG：适合图标、插图等小尺寸文件，支持阿尔法透明度设置
  3. GIF
  4. SVG：可以直接嵌入网页中，也可以作为资源引用；可以作为背景图，也可以作为内容图
  5. WebP：谷歌开发的一种新图片格式
  • Background-position不能一个关键字加一个长度值。
  • Box-shadow使用inset关键字可以为元素设置内阴影。
  • Object-fit属性可以像使用background-size一样，保持元素的宽高比：
    1. object-fit默认值为fill意味着图片内容会在必要时拉伸以填满容器，因此可能破坏宽高比
    2. cover和contain与background-size对应的关键字相同
    3. scale-down会自动从none和contain中选一个，哪个结果尺寸小就用哪个

内容布局：

  • 每一个评注的组件都是aside组件，然后对其使用绝对定位，再通过负外边距来移动元素。

  • 我们希望某元素总是与外部元素保持一定距离，但又不给任何元素设置大小，可以只指定其右、下、左边的偏移。

  • 若我们希望把两个元素垂直居中，需要让这两个元素都参照同一个“中心点”:vertical-align:middle

  • 伪元素会创建一个空格，导致内容向右偏移几个像素，若想实现真正意义上的居中，通过给伪元素应用外负边距-.25em可以抵消空格。

  • 通过Flexbox可以轻松实现多个元素垂直居中。

表单：

  • 字段名用label元素表示，用于给表单添加结构。

  • 在表单中设置border属性通常会去掉操作系统默认添加的边框，以及文本输入框的内阴影。

  • 使用buttom来创建按钮可以在里面嵌入其他元素来应用样式。

  • 可以使用placeholder属性提供示例输入。

  • 要实现更高级的表单的布局结构，要用到flexbox布局。

  • .field和.flexbox都是作为选择符的前缀。

  • 复选框可以使用无序列表来完成。

变换、过度与动画

  • 变换改变的是元素所在的坐标系统。

  • ::marker伪元素可以控制列表记号的样式，用counter-属性＋伪元素也可以。
    声明多个变换以后，如果想增加新变化，不能直接在原来的基础上添加，而要重新声明整套变换。

  • 变换动画最好使用transition-property指定需要过渡的元素，再对它们设置时间，这样表示过渡元素的时间是公共的。

  • 缓动由transition-timing-function属性来指定，关键字有：linear、ease-in开始慢后来快、ease-out开始快后来慢、ease-in-out两头慢中间快。

  • 关键帧中的属性没有任何特殊性，不允许添加!important。

  • 要定义透视，先得确定用户距离这个元素有多远。离得越近变化越大，离得越远变化越小。默认的距离是无穷远，因此不会发生明显的变化。

高级特效：

  • CSS Shapes包含两组新属性，一组用于设置影响盒子中内容的形状，另一组用于设置影响元素周边内容流的形状。

  • Shape-outside属性只能应用给浮动元素。

  • 通过shape-outside可以实现文本流环绕效果。
